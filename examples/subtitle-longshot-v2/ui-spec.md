# UI Spec（v2）

## 视觉主题
- 风格关键词：清爽功能感、移动端原生卡片、强调主流程 CTA。
- 主色：`#0f9d7a`（功能主操作）
- 强调色：`#ff8a3d`（进度与状态强调）
- 背景策略：浅绿灰渐变底 + 白色卡片层，形成信息层级。

## 移动端规格
- 平台：Mobile（默认 iOS 风格中性）
- 逻辑宽度：390（`max-width: 420px`）
- 安全区：顶部与底部均使用 `safe-area-inset-*`
- 页面骨架：顶部导航 + 内容滚动区 + 底部固定操作区
- 弹层策略：Bottom Sheet（处理页/预览页都支持）

## 页面与主流程
- `index.html`：功能入口、最近任务、主 CTA
- `process.html`：参数配置、处理进度、手动字幕区域调整
- `preview.html`：拼接预览、条目删改、导出入口
- `export.html`：导出预览、保存相册/系统分享
- `history.html`：历史任务追踪

主流程链路：
`index.html -> process.html -> preview.html -> export.html`

## PRD 映射矩阵
| PRD 需求点 | 页面/模块 | 状态 |
|---|---|---|
| 自动截取字幕区域图片 | `process.html` 字幕区域 + 手动调整 Sheet | 已覆盖 |
| 筛除字幕相同帧 | `process.html` 去重开关 + 处理阶段说明 | 已覆盖 |
| 组合成一张新的图片 | `preview.html` 拼接预览 + `export.html` 导出预览 | 已覆盖 |
| 用于发布社交网络或保存相册 | `export.html` 系统分享 / 保存相册按钮 | 已覆盖 |

## 交互状态
- 处理：未开始 / 处理中 / 完成 / 取消
- 条目：删除 / 还原
- 弹层：打开 / 关闭 / 应用
- 导出：保存成功提示 / 分享提示（原型示意）
